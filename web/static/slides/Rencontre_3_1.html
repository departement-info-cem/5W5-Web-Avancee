Fluent API

  • Permet de créer ou clarifier des relations entre des objets
  • Complète l’utilisation des propriétés de navigation
  • Dans cet exemple, on évite de faire un cascade delete, qui est le comportement par défaut pour une relation.

builder.Entity()
  .HasOne(m => m.PlayerDataA)
  .WithMany()
  .OnDelete(DeleteBehavior.NoAction);

builder.Entity()
  .HasOne(m => m.PlayerDataB)
  .WithMany()
  .OnDelete(DeleteBehavior.NoAction);
  • Fluent API va être présenté avec plus de détails plus tard dans la session
